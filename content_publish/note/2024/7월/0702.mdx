1. JSON을 body로 받거나 보낼 때 유효한 타입은 3가지다
	1. 숫자형(401003)
	2. 문자열("아무거나")
	3. 불리언형(true/false)
	따라서 아래와 같은 형태는 parse할 때 에러가 난다
	```
	{
		"code": 401001,
		"error": UNAUTHORIZED, // SyntaxError: Unexpected token 'U', ..." "error": UNAUTHORIZ"... is not valid JSON
		"message": "문자열"
	}
```
