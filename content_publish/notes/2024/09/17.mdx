null과 undefined의 차이
null은 빈 값을 할당한 상태(의도적, `typeof null === object` )
undefined는 자료형이 없는 상태(값이 할당되지 않음)
JSON에서 undefined는 지원하지 않는 타입.

JS || (OR operator) vs ?? (null-coalescing operator)
OR 연산자는 falsey 여부로 우측 값을 할당하고, null 중첩 연산자는 null이거나 undefined인지를 확인한다. OR 연산자가 null 중첩 연산자에 비해 좀 더 포괄적.

```javascript
    // OR operator can coerce 'defined' values
    "value"   || null;    // "value"
    0         || null;    // null
    false     || null;    // null
    ""        || null;    // null
    undefined || null;    // null
    
    // The null-coalescing operator will only coerce undefined or null
    "value"   ?? null;    // "value"
    0         ?? null;    // 0
    false     ?? null;    // false
    ""        ?? null;    // ""
    undefined ?? null;    // null
```
