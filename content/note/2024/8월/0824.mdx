수정기능
- 수정을 한다 -> 어딘가에 수정한 셀의 메타정보 배열로 저장 -> 하이라이트로 수정 상태 셀 렌더 -> POST 통신 후 하이라이트 모두 제거(일괄 삭제해도 되나?)
- 기존의 수정 -> rows 데이터를 useState에 넣고 handleChange 함수로 state를 업데이트하는 방식
- state 업데이트하는 방식은 그대로 유지함(어차피 새로고침하면 서버 기준 데이터로 초기화되기 때문)
- 그대신 handleState에서 셀을 찾아 업데이트하는 과정에 highlights 배열에 값을 추가하는 로직을 추가
