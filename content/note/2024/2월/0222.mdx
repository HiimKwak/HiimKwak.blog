- 종윤님한테 도메인 설명받음
- 오더/견적 삭제 기능 구현(완)
	- git flow: 아래와 같으며, 현재는 develop 서버 구현 미완성으로 local 자가 테스트 후 staging merge해 종윤님 테스트, 테스트 통과하면 서버 배포 후 클라 배포
	- 나중에 정비되면 develop에서 staging 테스트를 진행하고, staging 브랜치는 배포 전 테스트로 바뀔 것 같다
```
develop <- 만든 브랜치 (먼저 테스트) 

테스트 통과하면 

staging <- 만든 브랜치 (테스트 지금은 큰 의미 없지만, 나중엔 필요함) 

테스트 통과하면 

main <- 만든 브랜치
```
